# Taylor Made Esthetics — Spa Parties 💗✨

A pink & gold landing page for Taylor Made Esthetics' mobile kids spa parties and
teepee sleepovers, serving Pearland, TX & the greater Houston area.

Built with [Next.js 15](https://nextjs.org) (App Router) + [Tailwind CSS v4](https://tailwindcss.com),
optimized for mobile and local SEO (meta tags, Open Graph, JSON-LD LocalBusiness schema,
sitemap & robots).

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploying

The easiest path is [Vercel](https://vercel.com/new) — import this GitHub repo and
deploy with the default Next.js settings. When the final domain is decided, update
`siteUrl` in `app/layout.tsx` and the URLs in `app/sitemap.ts` / `app/robots.ts`.

## Content

- Party descriptions & add-on pricing live in `app/page.tsx`
- Brand colors & fonts are defined in `app/globals.css`
- Images are in `public/images/` (source flyers in `assets/`)
